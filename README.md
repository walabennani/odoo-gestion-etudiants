# Module Odoo : Gestion des Étudiants

## Description
Ce module Odoo permet de gérer les étudiants au sein d’un établissement scolaire ou universitaire.  
Il offre des fonctionnalités pour gérer les informations des étudiants, leurs classes, les notes et fournit un **dashboard pour visualiser les statistiques principales**.

---

## Fonctionnalités principales
- Ajouter, modifier et supprimer des étudiants
- Gestion des classes et affectation des étudiants
- Gestion des notes des étudiants
- Consultation des informations des étudiants, classes et notes
- Dashboard avec :
  - Total des étudiants
  - Total des classes
  - Total des notes
  - Moyenne générale des notes
  - Graphique de la moyenne des notes par classe
- Sécurité par groupes d’utilisateurs pour limiter l’accès aux données

---

## Technologies utilisées
- Odoo 17
- Python
- XML pour les vues
- PostgreSQL pour la base de données
